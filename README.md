# LinkedIn Revenue Intelligence Dashboard

A self-service analytics dashboard built to analyze LinkedIn's revenue 
trends across business segments and quarters.

## What it does
- Tracks annual revenue growth from 2022 to 2024
- Measures quarter-over-quarter growth rates
- Breaks down revenue by segment (Talent Solutions, Marketing Solutions, Premium Subscriptions)
- Surfaces key seasonal patterns and strategic insights

## Built with
- Python
- DuckDB (SQL)
- Pandas
- Streamlit

## How I built this
I built this project over 2 days while applying for the LinkedIn Revenue Intern role 
to get hands-on exposure to how a revenue analytics workflow operates.

The SQL layer (revenue.py) I worked through step by step and understood fully — 
writing queries for growth rates, segment mix, and quarter-over-quarter analysis 
using DuckDB inside Python.

For the dashboard (dashboard.py), I used Claude AI as a coding assistant to help 
structure the Streamlit layout, as some of the techniques were beyond my current 
level. I reviewed and understood the final output, but want to be transparent that 
this was a collaborative effort with AI tooling.

I believe using AI as a learning tool — not just a shortcut — is a skill in itself, 
and one that aligns with how modern revenue teams operate.

## Why I built this
To demonstrate initiative, a bias for action, and genuine interest in the 
LinkedIn Revenue Intern role before even getting an interview.
